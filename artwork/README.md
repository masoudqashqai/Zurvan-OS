# Artwork

The Zurvan OS emblem: the twins of boundless time circling each other. The **lion**
(gold, what endures) is the sun — the Persian sun-lion, with a radiant mane of twelve
flames, one for each hour. The **snake** (teal, what sheds its skin) is the moon — an
ouroboros swallowing its own tail around a crescent and a star, the cycle that renews
itself every boot. Day above, night below; each keeps the other's color as its eye —
the two seeds of the taijitu. Twelve diamonds mark the outer ring like a clock face:
Zurvan is time itself.

Like everything else in this repository, the artwork is **built from source**:
every file here is generated by [`generate.py`](generate.py) (stdlib Python, no
dependencies — the letterforms are hand-drawn paths, so no fonts are needed either).

| File | Use |
|------|-----|
| [`zurvan-logo.svg`](zurvan-logo.svg) / [`.png`](zurvan-logo.png) | The emblem — README, avatars, stickers |
| [`zurvan-icon-256.png`](zurvan-icon-256.png) | Favicon / app-icon size |
| [`zurvan-wordmark.svg`](zurvan-wordmark.svg) / [`.png`](zurvan-wordmark.png) | Emblem + ZURVAN lockup — release pages, social cards |
| [`zurvan-wallpaper.svg`](zurvan-wallpaper.svg) / [`-2560x1440.png`](zurvan-wallpaper-2560x1440.png) | Desktop wallpaper (the SVG rescales losslessly to any resolution) |

Regenerate everything with:

```sh
python3 generate.py
```

## Palette

| | Hex | Meaning |
|---|-----|---------|
| Gold | `#f6c65b` → `#b97b16` | the lion — permanence, `/data`, what endures |
| Teal | `#3ecfba` → `#0d7a70` | the snake — ephemerality, the OS reborn each boot |
| Night | `#161b2c` → `#0a0d16` | the background — boundless time |
