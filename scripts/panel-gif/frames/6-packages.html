ℹ What is this page?

A Zurvan package is a .tar.gz of static binaries plus a manifest, with a detached .sig beside it. Upload both below (or drop them in /data), then Install — the signature is verified before anything is unpacked, then files are linked into the standard paths on every boot.

Upload packages

Select a tarball and its .sig together (multi-select works) — or upload a whole zurvan-catalog-*.tar.gz release pack and every package inside is staged here with its .sig. Then click Install below.
