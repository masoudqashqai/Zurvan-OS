Jobs — the snake

ℹ What is this page?

The snake runs a throwaway script in a disposable sandbox — its own private tmpfs in a fresh mount namespace. The script gets an empty, writable system: it can create, delete, and scribble anywhere and none of it touches this machine. When it finishes, the sandbox evaporates.

Only three things come back: the exit status, the captured output (the log below), and any files the script copies into the $ARTIFACTS directory. Use it for a quick build, a one-off script, or trying a risky command safely.
